Bukan server kompetitif. Hanya sebuah dunia kecil tempat kamu bisa membangun rumah, bertani, dan nongkrong bareng teman setelah lelah beraktivitas seharian.

Kenapa Server Ini Dibuat?

Kami lelah dengan server besar yang penuh dengan Pay-to-Win, spam di chat, dan pemain yang saling menjatuhkan. Kami hanya ingin kembali ke akar Minecraft: Bermain bersama teman.

Server ini dikelola secara patungan dan sukarela. Tidak ada bos, tidak ada target. Kita semua di sini hanya ingin membangun desa kita sendiri secara perlahan.

Nilai & Aturan Kita

Hal-hal yang membuat komunitas kita nyaman.

Vanilla Santai

Tanpa plugin ribet. Main murni survival layaknya singleplayer, tapi bareng-bareng.

Saling Percaya

Tidak ada pencurian atau griefing. Jika kamu butuh item, cukup tanya di chat, pasti ada yang bantu.

Anti Pay-to-Win

Tidak ada VIP yang bikin overpower. Semua orang setara, uang donasi hanya murni untuk bayar sewa server bulanan.

Voice Chat

Kita pakai mod Simple Voice Chat. Jadi bisa langsung ngobrol santai saat ketemu teman di dalam game!
